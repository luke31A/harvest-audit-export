# Harvest Excel Export

Python script that pulls Harvest time entry data via API and exports a formatted Excel report.

## Usage
Run the .exe, enter a date range when prompted, get an Excel file in the output folder.

## Files
- `src/harvest_export.py` — main script
- `src/config.py` — API credentials (not committed)
- `dist/` — compiled .exe lives here
- `output/` — generated Excel files land here
